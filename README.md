## 📌 Overview
An end-to-end Machine Learning pipeline to predict automobile market prices. This project focuses on data cleaning, handling outliers, and building a high-performance regression model to achieve an accuracy of over **81%**.

---
## Project Structure
- `eda.ipynb`: Exploratory Data Analysis and visualization.
- `preprocessing.py`: Data cleaning and encoding functions.
- `modeling.py`: Model training and saving (`model.pkl`).
- `testing.py`: Making predictions on unseen data.
- `data/`: Folder containing `train.csv` and `test.csv`.

## 🗺️ Project Roadmap
To build this project, we followed these professional data science steps:

1.  **Exploratory Data Analysis (EDA):** Visualizing data distributions and correlations.
2.  **Data Preprocessing:** Handling missing values and removing unrealistic price outliers.
3.  **Feature Encoding:** Converting categorical car specs (Fuel type, Gearbox, etc.) into numerical data.
4.  **Model Training:** Using the **Random Forest Regressor** to learn from the data.
5.  **Evaluation:** Testing accuracy on unseen data to ensure reliability.
